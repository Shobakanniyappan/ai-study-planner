# AI-Powered Study Planner

An intelligent study scheduling application built with Python and Streamlit. This project is designed to help students optimize their preparation for university exams by calculating daily study requirements based on subject complexity and time constraints.

## ✨ Features
- **Dynamic Scheduling:** Adjusts study hours based on the remaining days before the exam.
- **Difficulty Weighting:** Prioritizes complex subjects (like Machine Learning or AI) over simpler ones.
- **User-Friendly UI:** A clean web interface for easy interaction.

## 🛠️ Technology Stack
- **Language:** Python 3.x
- **Frontend/Hosting:** Streamlit
- **Logic:** Custom Python Scheduling Algorithm
